# Code Book for Getting and cleaning data course project

You may want to read `README.dm` for background information.

## Data

The `tidy.txt` contains the data set with space separated values.

The header or the first row contains the names of the columns and the nexts rows contains the values.

## Identifiers

The dataset is composed by 79 averaged signal measurments for each subject and activity.

- `subject`
		An integer value that specifies the identifier of each subject. The range is: 1-30

- `activity`
		Type activity of each person, -String with 6 posibles values-:
		
		- `WALKING`: subject was walking (value `1`)
		- `WALKING_UPSTAIRS`: subject was walking upstairs (value `2`)
		- `WALKING_DOWNSTAIRS`: subject was walking downstairs (value `3`)
		- `SITTING`: subject was sitting (value `4`)
		- `STANDING`: subject was standing (value `5`)
		- `LAYING`: subject was laying (value `6`)
